CXXV. Program Execution Functions

Введение

Those functions provide means to execute commands on the system itself, and means to secure such commands.

Требования

Эти функции всегда доступны.

Установка

Для использования этих функций не требуется проведение установки, поскольку они являются частью ядра PHP.

Настройка во время выполнения

Данное расширение не определяет никакие директивы конфигурации в php.ini.

Типы ресурсов

Данное расширение не определяет никакие типы ресурсов.

Предопределенные константы

Данное расширение не определяет никакие константы.

Смотрите также

These functions are also closely related to the backtick operator. Also, while in безопасный режим you must consider the safe_mode_exec_dir directive.

proc_nice -- Change the priority of the current process
